Required Software: 
- JDK 1.5++

Preparation
- Extract the project zip to some destination. You will have the CignaProgrammingTest folder
- Sample case is already available with input file named "roman_in.txt" in CignaProgrammingTest folder.
- Enter command line and change current directory to the CignaProgrammingTest folder
 
How to compile
CignaProgrammingTest> javac src\com\codingtest\RomanCalculator.java -d bin

How to run (the output will be written to roman_out.txt file)
CignaProgrammingTest> java -classpath bin com.codingtest.RomanCalculator
